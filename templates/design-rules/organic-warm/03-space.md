# Space & Rhythm — Organic Warm

## 핵심 원칙

공간은 숨 쉬는 것이다. 넉넉하게, 여유롭게.
엄격한 그리드보다 유기적인 흐름이 중요하다.
콘텐츠가 자연스럽게 흘러가는 느낌을 만든다.

## Base Unit

**8px** 기준. 단, 유기적 느낌을 위해 비정수 배수도 허용한다.

```
8px   — 마이크로: 아이콘-텍스트 간격
12px  — 관련 요소 내부
16px  — 인풋 패딩, 기본 간격
20px  — 카드 내 텍스트 간 여백
28px  — 카드 내부 패딩 (최소)
40px  — 카드 내부 패딩 (넉넉)
48px  — 컴포넌트 간 간격
80px  — 섹션 간 간격
```

## 밀도 설정: Generous

Organic Warm은 **넉넉한 패딩**이 특징이다.

- 카드 패딩: 28~40px (좁으면 답답하다)
- 섹션 패딩: 80px 상하
- 버튼 패딩: 12px 28px (여유 있게)
- 텍스트가 컨테이너 벽에 붙지 않는다
- "너무 넉넉한가?" 싶으면 올바른 상태다

## 그리드

```css
/* 유연한 그리드 — 엄격하지 않다 */
display: grid;
grid-template-columns: repeat(12, 1fr);
gap: 24px;
max-width: 1120px;
margin: 0 auto;
padding: 0 32px;
```

그리드를 쓰되, **유기적 흐름이 우선**이다.
카드가 2열, 3열로 자연스럽게 배치되는 것이 중요하지
12컬럼 정밀 배치는 필요 없다.

## 최대 너비

```
전체 레이아웃: max-width 1120px
본문 텍스트: max-width 720px
카드 그리드: max-width 1120px
히어로 타이틀: max-width 800px
```

## 정렬 원칙

- 텍스트: 왼쪽 정렬 기본, 중앙 정렬도 허용 (히어로, CTA 영역)
- 카드: 균등 배치, 마지막 줄 왼쪽 정렬
- 이미지: 카드 안에서 둥근 모서리와 함께
- 전체적으로 **느슨한 정렬** — 1~2px 차이는 문제 아님

## 수직 리듬

```
H1 위: 64px
H2 위: 48px
H3 위: 32px
본문 단락 간: 20px
리스트 항목 간: 12px
```

## Claude에게

패딩이 부족해 보이면 늘려라. Organic Warm에서 "빡빡함"은 잘못된 상태다.
그리드에 억지로 맞추려 하지 마라. 자연스러운 배치가 우선이다.
