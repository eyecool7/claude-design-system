# Space & Rhythm — Luxury Minimal

## 핵심 원칙

여백은 가장 비싼 재료다.
Luxury Minimal에서 여백은 넉넉한 수준을 넘어 **과할 정도로** 크다.
요소 사이의 거리가 고급스러움을 만든다.

## Base Unit

**8px** 기준. 그러나 대부분의 값은 큰 배수다.

```
8px   — 마이크로: 아이콘-텍스트 간격
16px  — 관련 요소 내부
24px  — 인풋-라벨 간격
40px  — 카드 내부 패딩
80px  — 컴포넌트 간 간격
120px — 섹션 패딩 (최소)
160px — 히어로 섹션 상하 패딩
```

## 밀도 설정: Extreme Airy

Luxury Minimal은 **가장 넓은 여백**을 사용한다.

- 섹션 패딩: 최소 120px (더 커도 된다)
- 카드 패딩: 40px 이상
- 네비게이션과 첫 콘텐츠 사이: 최소 80px
- 컴포넌트 사이: 80px
- "이 공간이 낭비 아닌가?"라는 생각이 들면 올바른 상태다

## 그리드

```css
display: grid;
grid-template-columns: repeat(12, 1fr);
gap: 24px;
max-width: 960px;
margin: 0 auto;
padding: 0 80px;
```

## 최대 너비

```
전체 레이아웃: max-width 960px
본문 텍스트: max-width 600px
히어로 타이틀: max-width 720px
이미지 + 텍스트: max-width 960px
```

**960px** — 좁은 레이아웃이 집중도를 높인다.
1200px 이상의 넓은 레이아웃은 Luxury가 아니다.

## 정렬 원칙

- 텍스트: 왼쪽 정렬 기본, 중앙 정렬 허용 (히어로, 단일 CTA)
- 컴포넌트: 중앙 정렬 또는 비대칭 배치 (50:50이 아닌 40:60)
- 이미지 + 텍스트: 넓은 gap (80px)으로 분리
- 숫자: 오른쪽 정렬

## 수직 리듬

```
H1 위: 120px
H2 위: 80px
H3 위: 48px
본문 단락 간: 24px
섹션 전환: 120~160px
```

모든 수직 간격은 다른 Personality보다 **1.5~2배** 크다.

## Claude에게

여백을 줄이고 싶은 충동이 들면 줄이지 마라.
"공간이 비었다"는 것은 Luxury Minimal에서 최고의 칭찬이다.
max-width 960px을 넘기지 마라. 넓어지면 고급스러움이 사라진다.
